# Security Policy

## Supported versions

The latest published `1.x` release receives security fixes.

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Instead, report privately via
[GitHub Security Advisories](https://github.com/ASDAlexey/vitest-auto-spy/security/advisories/new).
You can expect an initial response within a few days. Once a fix is available it will be
released and the advisory published.

Thank you for helping keep the ecosystem safe.
